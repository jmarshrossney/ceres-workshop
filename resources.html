External Resources¶

Python:

  • Official Python 3 documentation

  • Official Package index

  • Geeks for Geeks Python tutorials

Git and GitHub:

To bring up the help pages from your terminal: use the --help option, e.g.

git diff --help

  • Github help pages

  • git commands reference, git cheat sheet, and a visual git cheat sheet

  • Github Desktop homepage

Conda and packaging:

  • Conda user guide and Conda cheat sheet

  • Mamba installation instructions

  • Conda-lock - if you really care about reproducibility and want to use Conda, you should use this instead of conda env export

  • Conda Forge - where you will find (almost) all of the packages you need

  • Poetry - an alternative way to manage dependencies and virtual environments which is (unlike Conda) Python-specific. I highly recommend using this over Conda if you work exclusively in Python

  • Flit documentation - we used this to build our own packages
